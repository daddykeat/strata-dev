# Strata Dev — Portfolio Website

A multi-page personal portfolio built with semantic HTML5 and mobile-first CSS,
created by **Keaton Moore** for the Module 6 Code-Along.

## Live site
🔗 **Published URL:** https://daddykeat.github.io/strata-dev/

## Pages
- `index.html` — Home / landing page
- `about.html` — Background, bio, and toolbox
- `projects.html` — Selected project work
- `contact.html` — Contact links (email, GitHub, portfolio)

## Tech stack
- HTML5 (semantic structure: `header`, `nav`, `main`, `footer`)
- CSS3 — mobile-first, with a `min-width: 720px` media query for tablet/desktop layouts
- Hosted on GitHub Pages

## Structure
```
/
├── index.html
├── about.html
├── projects.html
├── contact.html
├── styles.css
└── README.md
```

## Notes
- All pages share a single `styles.css` stylesheet.
- Navigation uses relative paths so links work locally and when published.
- The `viewport` meta tag is included on every page for responsive rendering.
